# Avaliação 01 - Implementação de Pilha

## Objetivo
Implementar uma Pilha (LIFO) com as operações:

- Mostrar topo
- Empilhar
- Desempilhar
- Mostrar todos os elementos

## Arquivo do código
O código está em:

```javascript
pilha.js
```

## Como executar
Executar no Programiz ou no console do navegador.

## Saída esperada
- HTML empilhado com sucesso
- CSS empilhado com sucesso
- JavaScript empilhado com sucesso
